## Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo.git
```

### 2. Open the repository in VSCode
```
code DEVSMap_to_Cadmium_Generator
```

### 3. (Optional) Edit the input files
```
Input files are located in the input folder.  Note that the parser is not fully implemented and only able to run simple examples at this time.
```

### 4. Run the parser and generate C++ Code for Cadmium
```
python ./DEVSMap_parser.py
```

### 5. Open a new WSL terminal in VSCode, and navigate to the output folder.
```
wsl
cd output
```

### 6. Build the executable
```
source build_sim.sh
```

### 7. Run the executable
```
./bin/Executable1
```
